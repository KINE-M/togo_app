# AppName
ToGoApp

- 行きたいところを登録できる、
- 行きたいところの位置情報を登録できる。 → Google Map と連携
- 行きたいところに行ったら、行ったところの感想を投稿（公開）できる。
- 公開されたすべての投稿の位置を地図上に表示する。
- ログイン機能
